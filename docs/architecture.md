# AWS Cloud Architecture Overview

This architecture demonstrates a scalable and secure AWS infrastructure.

## Services Used

- EC2 (Compute)
- S3 (Storage)
- VPC (Networking)

## Key Features

- Secure VPC configuration
- Public and Private Subnets
- Scalable EC2 deployment
- Static content hosting via S3
